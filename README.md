# Spendsense: Global Budget & Goal Simulator
**A data-driven financial planning tool for life in Canada (2026).**

Spendsense is a professional simulation tool designed to help diverse users-from single individuals to large families and international students-predict thir monthly savings and long-term financial health.

## Key Features
- **Persona-Based Logic:** Tailored budget defaults for Single Personas, Students, International Students, and Families.
- **Dynamic Family Scaling:** Automatically adjusts housing and grocery costs based on family size (2-8 members).
- **International Student Focus:** Includes mandatory health insurance tracking and 2026 proof-of-funds requirements.
- **Goal Forecasting:** Predicts exactly how many months it will take to reach savings goals based on current budget surplus.

## Tech Stack
- **Language:** Python
- **Framework:** Streamlit
- **Data Logic:** Predictive financial modeling and dynamic logic branching.
