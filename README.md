# SpendSense: 2026 Financial Simulation Lab

**An advanced economic modeling utility for real-time budget forecasting and investment analysis.**

SpendSense is a professional-grade financial laboratory designed to simulate personal budget performance under various 2026 economic scenaarios. The application utilizes a dynamic inflation-modeling engine and a multi-vector investment "What-If" calculator to project long-term capital growth.

## Techinal Features
- **Inflation Simulation:** Implements a real-time select-slider to forecast the impact of variable inflation rates (0% - 15%) on fixed monthly expenses.
- **Visual Analytics:** Features an interactive Plotly-driven Donut Chart for categorical expense distribution and structural budget visualization.
- **Compound Growth Engine:** Utilizes monthly compound interest algorithms to simulate 5-year investment returns based on user-defined growth rates.
- **Privacy Architecture:** Built on a zero-data persistence model, ensuring no personal financial information is cached or stored externally.

## Tech Stack
- **Python 3.10+**
- **Streamlit** (User Interface)
- **Plotly Express** (Data Visualization)
- **Pandas** (Data Structures)
